Introduction
---------------------
Selectively disable node listings on taxonomy term pages on a vocabulary basis. 
Once disabled view blocks or views embedded using EVA: Entity Views Attachment 
can be used instead.

Configuration
---------------------
Disable term listings for a particular vocabulary by going to the edit page.
For example, "structure/taxonomy/[term_machine_name]/edit".
