Introduction
---------------------
Selectively disable node listings on taxonomy term pages on a vocabulary basis. Once disabled view blocks or views embedded using <a href="http://drupal.org/project/eva">EVA: Entity Views Attachment</a> can be used instead.

Configuration
---------------------
Disable term listings for a particular vocabular by going to the edit page (structure/taxonomy/another_test/edit).
